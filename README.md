# WsaPartner

### 一个ADB工具，采用WinUI3第三个预览版结合MVVM模式开发，主要是用来安装apk应用到Wsa系统上。
### 目前开源协议是MIT的，但是参考了一些GPL-3.0的代码，后期为了防止传染应该会全部移除。
### 欢迎大家前来贡献代码，目前还处于开发中的状态。
### 应用参考了一个叫做[WsaToolbox](https://github.com/makazeu/WsaToolbox)的做了一些功能，他没开源只能抄他的UI了。

## 参考项目仓库

- [Windows Community Toolkit](https://github.com/windows-toolkit/WindowsCommunityToolkit/tree/winui)

- [SharpAdbClient](https://github.com/quamotion/madb)
